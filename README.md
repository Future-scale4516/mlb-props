# MLB Prop Analyser

## Run Locally
1. Install Python 3.10+ from https://python.org
2. pip install -r requirements.txt
3. streamlit run app.py
4. Browser opens at http://localhost:8501

## Deploy Free on Mobile (Streamlit Cloud)
1. Push this folder to GitHub
2. Go to https://streamlit.io/cloud -> New App
3. Connect repo, set app.py as entry point -> Deploy
4. Bookmark the URL on your phone

## Daily Workflow
- Step 1: Add games (pitchers, park, weather)
- Step 2: Enter confirmed lineups
- Step 3: Hit Score -> ranked results by market
- Download CSV or screenshot top picks
